# Blender Utilities
Collection of small and useful Blender scripts.

## Installation:
1. Go into your "C:\Users\USERNAME\AppData\Roaming\Blender Foundation\Blender\BLENDER_VERSION\scripts\startup" folder 
2. Copy your desired python script into it
3. Restart Blender
4. Search in your F3 and look for the name of the script
